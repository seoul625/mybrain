# 📋 작업 브리프

**원 명령:** ppt나 pdf파일 자료가 있는데..그걸 볼수가 없어?

## 요약
사용자가 PPT나 PDF 형태의 자료를 시스템이 직접 열람하거나 내용을 분석할 수 있는지 문의함. 이는 시스템의 기능적 한계와 데이터 처리 범위를 설명하는 것이 목표임.

## 분배
- **📱 영숙**: 사용자에게 시스템의 현재 기능적 한계(직접 파일 열람 불가능)를 명확하게 설명하고, 대신 파일 내용을 텍스트로 복사/붙여넣기 하거나, 파일의 핵심 요약/목차만 제공해 달라고 구체적으로 요청하여 작업을 진행할 수 있도록 안내한다.
