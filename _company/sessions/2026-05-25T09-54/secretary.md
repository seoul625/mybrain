# 📱 영숙 — 사용자가 전달한 모든 이미지 자산(J:\Ai Visual Lab관련\homepage\image)과 이미지 프롬프트 파일 목록을 취합하여 '자산 목록(Manifest)'을 작성해 주세요. 파일명, 경로, 용도(홈페이지 이미지), 그리고 가장 중요한 '원본 프롬프트'를 명확하게 정리해야 합니다. 또한, 이 자산들을 기반으로 어떤 종류의 마케팅 콘텐츠(예: 랜딩 페이지 섹션, 블로그 게시물, 소셜 미디어 셋)를 제작할 수 있는지 초기 아이디어 3가지를 제안해야 합니다.

네, 알겠습니다. 현재 가지고 계신 이미지 자산들을 체계적으로 정리하여 **'자산 목록(Asset Manifest)'**을 만들고, 이를 기반으로 초기 **콘텐츠 전략 아이디어**를 도출하는 것이 목표이시군요.

최대한 정확하고 활용도 높은 결과물을 만들기 위해서는, 먼저 **어떤 파일들이 포함되어 있는지 목록(File List)을 확인**해야 합니다.

**💡 다음 단계 진행을 위해, 이미지 파일들이 있는 폴더의 파일 목록 전체 또는 파일의 메타데이터(파일 이름, 파일 크기, 생성일 등)를 제공해 주시면 감사하겠습니다.**

---

### 📋 파일 목록을 받는 즉시 진행할 프로세스 (예상 결과물 구조)

파일 목록을 받는 대로, 아래와 같은 3단계의 보고서 형태로 정리하여 드리겠습니다.

#### 1단계: 🖼️ 이미지 자산 목록 (Asset Manifest)
*   **목적:** 모든 자산의 개요와 식별 정보를 정리합니다.
*   **포함 내용:**
    *   **파일 이름:** (예: `product_001.jpg`)
    *   **카테고리 분류:** (예: 제품 사진 / 라이프스타일 / 인물)
    *   **핵심 키워드:** (예: 민트색, 미니멀리즘, 야외활동)
    *   **추정 역할:** (예: 메인 배너용 / 상세 페이지용 / SNS 홍보용)

#### 2단계: 📝 자산별 상세 메타데이터 및 프롬프트 (Metadata & Prompt Guide)
*   **목적:** 각 이미지를 어떻게 활용할지 구체적인 가이드라인을 제시합니다.
*   **포함 내용:**
    *   **이미지 분석:** 해당 이미지에서 가장 돋보이는 점을 텍스트로 설명합니다.
    *   **추천 활용 시나리오:** 이 이미지가 어떤 페이지(랜딩 페이지, 제품 상세 페이지, 배너 등)에 가장 적합한지 제안합니다.
    *   **마케팅 문구 (Copywriting Suggestion):** 해당 이미지를 사용할 때 어울리는 헤드라인이나 짧은 설명 문구 초안을 2~3가지 제안합니다.

#### 3단계: 🚀 초기 콘텐츠 전략 및 활용 방안 (Content Strategy Roadmap)
*   **목적:** 개별 이미지를 넘어, 전체 자산을 활용한 큰 그림의 마케팅 방향을 제시합니다.
*   **포함 내용:**
    *   **테마별 그룹화:** 자산을 묶어 만들 수 있는 3~5개의 핵심 캠페인 테마를 제안합니다. (예: '가을맞이 리프레시', '재택근무 필수템')
    *   **채널별 배포 가이드:** 각 테마를 인스타그램(릴스/피드), 블로그, 광고 배너 등 매체별로 어떻게 변주하여 활용할지 로드맵을 제시합니다.
